# 📊 Excel Analyzer Dashboard

A powerful web-based tool to analyze Excel files directly in your browser — no data upload, no backend, full privacy.

---
## 📸 Excel Analyzer 

![App Excel Analyzer](images/Excel Analyzer.png)

## 🚀 Features

### 📁 File Upload & Processing
- Supports:
  - `.xlsx`
  - `.xls`
  - `.csv`
  - `.ods`
- Drag & Drop or manual upload
- 100% client-side processing (no data leaves your device)

---

### 📊 Interactive Dashboard
- Instant KPIs:
  - Total Rows
  - Total Columns
  - Detected Formulas
  - Empty Values
  - Numeric Columns

- Dynamic charts:
  - Bar Chart
  - Line Chart
  - Pie Chart

---

### ⚡ Formula Analysis
- Extracts all formulas from the sheet
- Categorizes formulas by type
- Displays computed values

---

### 🔢 Column Statistics
- Automatic numeric column detection
- Per-column stats:
  - Average
  - Median
  - Min / Max
  - Standard Deviation
  - Sum

---

### 💰 Advanced Accounting Analysis
- Financial breakdown:
  - Positive values
  - Negative values
  - Net balance

- Detects:
  - Outliers (based on standard deviation)
  - Rounded numbers (e.g. multiples of 1000)
  - Zero values

- Balance validation insights

---

### 🔁 Duplicate Detection
- Full row duplicates
- Column-level duplicates
- Detailed duplicate insights

---

### ⚠️ Smart Alerts System
- Missing data warnings
- Duplicate data alerts
- Anomaly detection
- High variance warnings
- Financial inconsistencies

---

### 🏆 Performance Insights
- Top 10 largest transactions
- Ranking system
- Positive vs negative highlighting

---

### 📅 Time-Based Analysis
- Monthly aggregation
- Automatic date detection
- Combined count + value charts

---

### 📋 Data Table
- Pagination
- Live search
- Clean table view

---

### 📥 Export Analysis
Export results to a new Excel file including:
- Statistics
- Accounting analysis
- Raw data
- Formulas
- Duplicates

---

## 🧠 Data Quality Score

Automatically calculates a smart score based on:
- Data completeness
- Uniqueness (duplicates)

---

## 🛠️ Tech Stack

- Vanilla JavaScript
- HTML5 + CSS3
- SheetJS (XLSX)
- Chart.js

---

## 🔒 Privacy First

✅ No server  
✅ No data upload  
✅ Fully local processing  

Your data never leaves your device.
